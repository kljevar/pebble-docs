# Installation

## npm

```bash
npm install pebble-design-system
```

## yarn

```bash
yarn add pebble-design-system
```

## pnpm

```bash
pnpm add pebble-design-system
```

## Peer dependencies

Pebble requires React and ReactDOM as peer dependencies:

```bash
npm install react react-dom
```

## Stylesheet

Import the Pebble stylesheet at your app entry point:

```tsx
import 'pebble-design-system/style.css'
```

This includes all design tokens and component styles.
