# Getting Started

Pebble is a React + TypeScript component library that brings the iOS 26 Liquid Glass aesthetic to the web.

## Prerequisites

- React 18+
- TypeScript 5+

## Install

```bash
npm install pebble-design-system
```

## Setup

Import the stylesheet once at your app root:

```tsx
import 'pebble-design-system/style.css'
```

Then use any component:

```tsx
import { Button, Card, Badge } from 'pebble-design-system'
```
