# Pebble

A React + TypeScript component library inspired by iOS 26 Liquid Glass.

## Features

- 11 production-ready components
- iOS 26 Liquid Glass aesthetic — frosted surfaces, spring animations, iridescent gradients
- Built with CSS Modules and design tokens
- Full TypeScript support

## Quick Start

```bash
npm install pebble-design-system
```

```tsx
import { Button } from 'pebble-design-system'
import 'pebble-design-system/style.css'

export default function App() {
  return <Button>Hello Pebble</Button>
}
```
